La interfaz de usuario

La interfaz de usuario es el punto entre el humano y la computadora/dispositivo ya que este nos ofrece información que los humanos entendemos y desde allí enviamos información u órdenes que las maquinas pueden procesar.

Pueden existir varios tipos de interfaces de usuario, entre las virtuales, físicas y auditivas cada una de ella debe mostrar información a su manera y de esta dependerá las ordenes que deben ejecutar posteriormente

Ejemplo de interfaces de usuario

  • Mouse de un equipo de computo
  • Control remoto de RF
  • Realidad Virtual
  • Cajeros Automáticos
  • Velocímetro
