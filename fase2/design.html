Proposito

El propósito de este este proyecto se basa en la necesidad de
elaborar recursos educativos digitales para dinamizar las estrategias de
aprendizaje que constituirá de Objetos virtuales de

La interfaz de usuario

La interfaz de usuario es el punto entre el humano y la computadora/dispositivo ya que este nos ofrece información que los humanos entendemos ydesde allí enviamos información u órdenes que las maquinas pueden procesar.

Pueden exixtir varios tipos de interfaces de usuario, entre las virtuales, fisicas y auditivas cada una de ella debe mostrar información a su manera y de esta dependera las ordenes que deben ejecutar posteriormente

Ejemplo de interfaces de usuario

  • Mouse de un equipo de computo
  • Control remoto de RF
  • Realidad Virtual
  • Cajeros Automáticos
  • Velocímetro

Interacción Humano Ordenador

La Interacción Humano Computador ó HCI es una diciplina que estudia cono es que la humanidad interactúa con las computadoras.

Cuando se lanzaron las computsdoreas personales svio reflejado como la tecnología se acopla de una manera progresiva al trabajo de cada persona , facilitando tareas y funciones en su dia a dia.

Esta disciplina de interaccion abarca más disciplinas como:

  • Informática
  • Inteligancia Artificial
  • Lingüistica
  • Filosofía
  • Antropología
  • Ciencia cognitiva
